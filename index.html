User Guide: Outlook

Step by step user guide on how to Import or Update Outlook Email Contact List


Step: 1

Download and Open the .zip contact file.

Step: 2

Extract (Drag and Drop) the file to Desktop (or any preferred location).


Step: 3

Next go back to Outlook, click on File > Open > Import.

Step: 4

This window should now pop up, click Next to continue.


Step: 5

Select Outlook Data File (.pst), to continue.

Step: 6

Click Browse and locate the contact list file AND MAKE SURE

  • “Replace duplicates with items imported” is selected.

before clicking Next.


Step: 7

Make sure “Import items into the same folder in:” is ticked and your email is selected then click Finish.

Step: 8

The email contacts list should now be imported to your address book when you click “To” on a new email window. or Home > Address Book.
